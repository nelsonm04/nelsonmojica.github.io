LanguageBuddy

Your personal companion for practicing new languages

Project Overview

LanguageBuddy is a mobile application designed to help users practice new languages through real conversations, session scheduling, and progress tracking.

Features

  • User authentication and profiles
  • Buddy discovery
  • Chat system with conversation history
  • Session scheduling
  • Ratings and progress tracking

Technology Stack

  • Android (Jetpack Compose)
  • Room Database
  • MVVM Architecture
  • GitHub Pages
